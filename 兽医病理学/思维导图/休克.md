# 概念
- 微循环有效灌流量不足
- 各组织器官缺血，缺氧，代谢紊乱，细胞损伤
# 休克的原因及类型
## 原因
### 心源性休克cardio-genic shock
- 心梗
- 心肌炎
- 急性心包积液
- 感染性休克
### 低血容量性休克hypo-volemic shock
- 失血/脱水导致血容量减少
### 血流分布异常性休克 blood mal-distribution shock
- 感染性休克 septic shock
- 过敏性休克 anaphylactic shock
- 神经源性休克 neurogenic shock
## 类型
### 心源性休克
- 心肌梗死 myo-cardial infarction
- 室性心动过速
- 纤颤 fibrillation
  - 房颤
  - 室颤
- 扩张性或肥厚性心肌病
### 低血容量性休克
- 低排高阻，冷休克
- 血容量降低10%
  - 微循环灌注压力不足
- 血容量降低35%-45%
  - BP&CO $\downarrow \downarrow$
### 血流分布异常性休克
- 血液容量正常，有效循环血量$\downarrow$（外周组织淤血）,外周阻力$\downarrow$
- 过敏性休克
  - I型变态反应
    - 全身血管舒张，毛细血管通透性$\uparrow$
- 神经源性休克
  - 中枢神经系统外伤/恐惧
  - 血管运动中枢自主放电-->血管扩张
- 败血性休克
  - 炎症因子大量释放
    - 血管扩张
# 血管生理
## 血管类型及功能
## 微循环构成模式
- 直捷通路
- 动-静脉短路
  - 感染性休克/中毒性休克时，皮肤表面动静脉吻合支大量开放 “**暖休克**”
- 迂回通路（真毛细血管）
## 微循环血液灌流调节
- 代谢产物反馈调节
- 神经调节
  - 儿茶酚胺$\uparrow$
    - 血管收缩，动静脉短路开放
- 体液调节
  - 收缩
  - 舒张
# 休克的过程
## 非进行期/休克代偿期 少灌少流，灌少于流
### 微循环变化特点
- 微循环小血管持续收缩
- 毛细血管前阻力>后阻力
- 关闭的毛细血管床
### 发生机制
- 缩血管
### 代偿意义：维持有效循环血量和血压/保证心脑血液供应
- 自我输血
- 自我输液
- 心输出量$\uparrow$
- 总外周阻力$\uparrow$
- 脑血管扩张
- 冠状动脉扩张
### 临床表现
- 少尿
- 面色苍白，四肢冰冷
- 出冷汗
- 烦躁不安
## 进行性休克期/微循环淤血期 灌而少流，灌大于流
### 微循环变化
- 微循环血管床大量开放 有效循环血量急剧$\downarrow$
- 血细胞黏附，集聚 血流阻力进行性$\uparrow$
- 血压进行性$\downarrow$
- 心脑灌流不足
### 发生机制
- 微循环血管对缩血管物质反应性$\downarrow$
- 局部扩血管物质增多
- 血细胞黏附，集聚
### 临床表现
- 神志淡漠
- 少尿/无尿
- 发绀/花斑（淤血）
## 不可逆休克期/休克难治期/微循环衰竭期 不灌不流，灌流停止
### 微循环变化
- 微循环血管麻痹
- 微血栓形成
- 弥散性血管内凝血DIC
### 临床表现
- 浅表静脉严重萎塌，静脉注射困难
- 心音低弱，脉快而弱
- 血压显著下降，给予升压药也难以恢复
- 呼吸不规则
- 无尿
- 皮下瘀斑，点状出血
- 多器官系统衰竭
# 器官功能障碍 impairment of organic function
## 肺功能障碍 respiration failure
- 休克肺
  - 肺水肿
  - 局灶性肺不张
  - 肺充血
  - 出血
  - 水肿
  - 微血栓形成
  - 肺泡内透明膜形成
## 肾功能障碍 renal failure
### 早期--急性功能性肾衰
- 血流灌注$\downarrow$
  - GFR$\downarrow$
    - 少尿
### 晚期--急性器质性肾衰
- 持续缺血&微血栓
  - 急性肾小管坏死
    - 少尿
## 心功能障碍 inpairment of cardiac function
- 冠脉血流量$\downarrow$
- 酸中毒
  - 高钾血症
    - 心肌收缩力$\downarrow$
- 心肌内DIC
- 毒性因子
## 多器官功能障碍综合征 MODS multiple organ disfunction syndrome
### 速发单相型
### 迟发双向型
# 防治原则
- 治疗原发病
- 改善微循环
  - 扩容
  - 血管活性药物
  - 防治DIC
- 纠正酸中毒
- 强心
